# Adamantine Android Client — README

This folder contains the **Android** version of the DigiByte Adamantine Wallet, built using **Kotlin + Jetpack Compose**.

## 1. Purpose
The Android client provides:
- Full Adamantine functionality  
- Strong mobile security  
- Native access to biometrics  
- Enigmatic chat integration  
- DD mint/redeem flows  
- Shield-layer diagnostics  

## 2. Technology Stack
- Kotlin  
- Jetpack Compose  
- AndroidX Security  
- WorkManager (background tasks)  
- WebRTC (future messaging relays)  

## 3. Feature Overview (Skeleton)
- Account dashboard  
- Send / Receive DGB  
- Mint / Redeem DD  
- Enigmatic chat interface  
- Shield status overlay  
- Guardian verification steps  
- Settings & Identity
