# Adamantine Web Client — README

This folder contains the Progressive Web App (PWA) version of the DigiByte Adamantine Wallet.

## 1. Purpose
The web client provides:
- Cross‑platform wallet access  
- Browser‑based DD mint/redeem  
- Enigmatic messaging integration  
- Shield diagnostics (Sentinel → DQSN → ADN → Adaptive Core)

## 2. Technology Stack
- TypeScript  
- WebAssembly (future PQC containers)  
- PWA offline caching  
- WebCrypto API  

## 3. Features (Skeleton)
- Account view  
- DGB send/receive  
- DD mint/redeem  
- Enigmatic chat UI  
- Shield status overlay  
- Settings + diagnostics  

## 4. Notes
The web client is fully optional.  
Mobile native apps provide the strongest security profile.
